1. Numeri Interi (Interi con segno)
Questi tipi rappresentano numeri senza virgola.
Alias C#	Tipo .NET	Dimensione	Intervallo (Approssimativo)
byte	System.Byte	8 bit	da 0 a 255
sbyte	System.SByte	8 bit	da -128 a 127

ushort	System.UInt16	16 bit	da 0 a 65.535
short	System.Int16	16 bit	da -32.768 a 32.767

uint	System.UInt32	32 bit	da 0 a 4,2 miliardi
int	System.Int32	32 bit	da -2,1 miliardi a +2,1 miliardi

ulong	System.UInt64	64 bit	da 0 a (18000 miliardi di miliardi)
long	System.Int64	64 bit	Molto grande (9000 miliardi di miliardi)

UInt128	System.UInt128	128 bit Estremamente grande
Int128	System.Int128	128 bit	Estremamente grande



3. Numeri a virgola mobile (Floating-Point)
Usati per calcoli scientifici o valori frazionari.
Alias C#	Tipo .NET	Dimensione	Precisione / Utilizzo
half	System.Half	16 bit	Bassa precisione (Machine Learning/GPU)
float	System.Single	32 bit	Precisione singola (Grafica/Giochi)
double	System.Double	64 bit	Precisione doppia (Standard scientifico)
decimal	System.Decimal	128 bit	Alta precisione finanziaria (Nessun errore di arrotondamento binario)

4. Testo e Logica
Alias C#	Tipo .NET	Descrizione
bool	System.Boolean	Rappresenta true o false.
char	System.Char	Un singolo carattere Unicode (16-bit).

5. Tipi Nativi (Dipendenti dall'architettura)
Questi tipi cambiano dimensione in base al processore (32 o 64 bit). Sono fondamentali per l'interoperabilità con C++ o la gestione della memoria ad alte prestazioni.
nint (System.IntPtr): Intero con segno di dimensione nativa.
nuint (System.UIntPtr): Intero senza segno di dimensione nativa.


CTS CLS CLR

CLR (Common Language Runtime): È il "motore" che esegue il codice.
CTS (Common Type System): È il "regolamento" sui tipi che il CLR segue.
CLS (Common Language Specification): È un sottoinsieme del CTS. Poiché il CTS contiene tantissime funzioni, non tutti i linguaggi le supportano tutte. Il CLS definisce un set minimo di regole a cui tutti i linguaggi devono attenersi per garantire la massima compatibilità (ad esempio, il CTS supporta i tipi senza segno come uint, ma il CLS consiglia di non usarli come pubblici perché alcuni linguaggi potrebbero non capirli).
